# -_project
간단한 게임을 만들어보고 싶었는데 포커(텍사스 홀덤)보다는 룰이 더 단순하고 구현하기 편할 것 같은 인디언 포커를 만들어보려고 했다. 처음에 트럼프 카드를 시각화하는 게 가능할까 고민했는데 실제 사진으로는 힘들거같아서 문자 배열로 만들어서 시각화하기로 생각하고 만들기 시작했다. 단순한 인디언 포커도 재밌겠지만 영화'타짜'가 생각나서 '밑장뺴기'라는 요소를 추가했다. 룰은 단순하다. 오직 상대방의 카드만 볼 수 있고 내 카드가 상대보다 높으면 이기는 게임이다. 처음은 플레이어가 덱을 섞고 그 이후부터는 승자가 덱을 섞는데, 밑장빼기를 하면 상대의 카드와 내 카드를 선택해서 세팅할 수 있다. 성공률은 30퍼센트인데, 생각보다 성공이 잘 안된다고 느꼈고 연속 성공은 필자도 성공 못했다.(걸리면 바로 손목이 날라가고 게임이 지므로 주의해서 사용 바람.)
